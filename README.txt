This repository contains code for a simple web page that allows users to generate gradient backgrounds by selecting colors or clicking a button to generate random colors.

## Features

- Allows users to select two colors using color pickers.
- Updates the background of the page with a gradient based on the selected colors.
- Includes a button to generate random colors for the gradient background.

## Usage

1. Open the `index.html` file in a web browser.
2. Use the color pickers to select your desired colors for the gradient background.
3. Alternatively, click the "Click me" button to generate random colors.
4. The background will update automatically as you select colors or click the button.

## Dependencies

This project does not have any external dependencies. It utilizes vanilla JavaScript for functionality.
